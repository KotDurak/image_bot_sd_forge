# 🎨 Telegram Forge Bot

Простой бот для удалённой генерации изображений через [Stable Diffusion WebUI Forge](https://github.com/lllyasviel/stable-diffusion-webui-forge).

## ✨ Возможности

- 🖼️ Генерация картинок по текстовому описанию
- 🎨 Выбор моделей из установленных в Forge
- 🎭 Пресеты стилей для быстрого старта
- 🔒 Контроль доступа по whitelist
- 📱 Удобное меню с inline-кнопками
