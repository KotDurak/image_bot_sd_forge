# 🎨 AI Generation Bot
> Telegram-бот для генерации изображений через Stable Diffusion (Forge API).  
> Умная очередь, монетизация, рекламная система и прозрачная аналитика.

## ✨ Возможности
- 🖼 **Генерация** по промпту с выбором моделей и пресетов
- 🔄 **Асинхронная очередь** с батчингом, автовосстановлением и переключением моделей
- 💰 **Система кредитов**: бесплатные лимиты, покупка генераций, возврат при ошибках
- 📢 **Рекламная система**: покупка показов, ротация, автоматическая деактивация
- 📊 **Отчётность**: `/ad_report` с пагинацией, `JOIN`-логика, экспорт в CSV
- 📥 **Импорт рекламы**: загрузка `.csv`/`.json` прямо в чат с валидацией
- 🛡 **Безопасность**: приватность данных, анонимизация в отчётах, транзакционные операции
- 📜 **История**: логирование генераций, аудит сбросов лимитов

## 🛠 Технологический стек
- `Python 3.10+` | `asyncio`
- `python-telegram-bot v20+`
- `aiosqlite` (асинхронная БД)
- `Stable Diffusion Forge API`
- Кастомный `GenerationQueue` с умным батчингом

## 🚀 Быстрый старт
```bash
git clone <repo-url>
cd bot_for_sd
python -m venv .venv && source .venv/bin/activate  # или .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env  # заполни токены и настройки
python main.py
```

## ⚙️ Конфигурация (.env)

Создайте файл `.env` в корне проекта и заполните его:

```env
# --- Основные настройки ---
TELEGRAM_TOKEN=...        # Токен Telegram-бота
ADMIN_USER_ID=...         # ID администратора
FORGE_URL=http://localhost:7860  # URL сервиса генерации / API

API_USER=...              # Логин для внешнего API
API_PASS=...              # Пароль для внешнего API

MODE=PROD                 # Режим работы: PROD или DEV


# --- Реклама и отчёты ---
ADS_ENABLED=true          # Включить рекламу
ADS_FOR_FREE_ONLY=false   # Показывать рекламу только бесплатным пользователям
ADS_PAID_CHANCE=0.1       # Вероятность показа рекламы платным (0.0 - 1.0)

AD_REPORT_ANONYMIZE=true  # Анонимизировать отчёты
AD_REPORT_SALT=change_me_in_prod  # Соль для хеширования (обязательно сменить в PROD)
```

## 📂 Архитектура проекта

| Папка / файл | Назначение |
|-------------|-----------|
| **`main.py`** | Точка входа, регистрация хендлеров, запуск приложения |
| **`handlers/`** | Команды, коллбэки, управление рекламой, обработка ошибок |
| **`services/`** | Очередь задач, рендер рекламы, планировщики, импортеры |
| **`db/`** | Ядро `aiosqlite`, SQL-миграции, методы доступа к данным |
| **`models/`** | Квоты, пресеты, история генераций |
| **`utils/`** | Логирование, вспомогательные функции (хелперы) |
| **`config.py`** | Константы, значения по умолчанию, пути |

## 📈 Монетизация
Бот поддерживает продажу показов рекламы. Рекламодатель покупает N показов → бот автоматически показывает рекламу после успешной генерации, списывая счётчик. Отчёты доступны админу через /ad_report <id>. Данные экспортируются в CSV с опциональной анонимизацией user_id.
